# MIND Diet App
A React + Vite + TailwindCSS v4 PWA for tracking Mediterranean-DASH Intervention for Neurodegenerative Delay (MIND) diet.

## Features
* Daily food tracking (good and bad foods)
* Weekly scoring based on MIND diet criteria
* Weekly history report with scores and counts
* Reminders with Notifications API
* PWA support (offline, install to home screen)

## Running the App
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`

## Building for Production
1. Build the app: `npm run build`
2. Preview the build: `npm run preview`
